# Cosechador de Limón de Tambogrande

## Descripción

Proyecto desarrollado como videojuego educativo basado en la producción y clasificación de limones de Tambogrande, Piura.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* PHP
* MySQL

## Funcionalidades implementadas

### Nivel 1

* Movimiento de canasta mediante teclado.
* Captura de limones.
* Sistema de puntajes.
* Cambio automático de nivel.

### Nivel 2

* Clasificación de limones.
* Arrastrar y soltar (Drag and Drop).
* Cajas de clasificación:

  * Tipo A
  * Tipo B
  * Descarte

### Funciones adicionales

* Pantalla de inicio.
* Registro de nombre del jugador.
* Cronómetro.
* Pantalla de resultados.

## Estado del proyecto

Versión funcional para demostración académica.
